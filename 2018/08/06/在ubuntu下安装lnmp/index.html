在Ubuntu下安装LNMP

Published on 星期一, 八月 6日 2018 at 11:53
Last updated on 星期一, 八月 6日 2018 at 16:46

LNMP介紹

LNMP是指在Linux系統下Nginx + Mysql + PHP的一種網站伺服器架構

本篇文章要教大家如何使用LNMP建置WordPress

環境:Ubuntu 16.04.4 LTS

1.安裝 Nginx 網頁伺服器

apt-get install nginx

安裝完後在瀏覽器輸入機器ip，看到以下畫面即安裝完成

2.安裝 Mysql 資料庫

apt-get install mysql-server

安裝過程中會彈出視窗提示輸入預設root帳戶的密碼，輸入密碼即可

安裝完後使用以下指令登入mysql:

mysql -u root -p

建置資料庫 

create database test;

以下指令可查看目前有的資料庫名稱

show databases;
